# Assets De Marca

Esta pasta existe para preparar o `Guarita` para multiplas marcas sem fork de codigo.

## Direcao recomendada

- manter uma pasta por marca;
- padronizar nomes de arquivo por tipo;
- referenciar esses assets a partir da configuracao de branding.

## Exemplo de estrutura futura

```text
assets/brands/
  rafiels/
    logo-primary.png
    logo-developer.png
    icon.png
    splash.png
  cliente-a/
    logo-primary.png
    logo-developer.png
    icon.png
    splash.png
```

## Observacao

No primeiro passo do whitelabel, o app ainda usa os assets padrao atuais:

- `assets/logo-v8.png`
- `assets/Logo-Rafiels.png`

Esta pasta foi criada para orientar a proxima etapa de organizacao por marca.
